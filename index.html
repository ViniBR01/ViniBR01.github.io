Let's Play Loteria!

This project was created to reproduce and expand the original Google doodle from 2019 that offers an original version of the game of loteria.

Improvements:

  • Allows for a larger number of simultaneous players
  • Connects players using a 4 letters code instead of link
  • Adds more options, such as selecting the game mode between random and full table

See the original game here: google.com/loteria
